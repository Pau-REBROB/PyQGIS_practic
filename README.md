# PyQGIS_practic
Aquest repositori pretén posar per escrit les diferents funcionalitats pràctiques de PyQGIS i posar-ho en pràctica amb la manipulació de dades, representació gràfica, anàlisi espacial i automatització de tasques.

Els scripts es van actualitzant a mesura que s'aconsegueix millorar el codi.

Documentació oficial: https://docs.qgis.org/3.44/es/docs/pyqgis_developer_cookbook/index.html

Dades extretes d'Open Data BCN sota llicència CC-BY 4.0 (https://opendata-ajuntament.barcelona.cat/ca/condicions-us).



Tots els scripts estan escrits sobre la base que s'utilitzen a la consola Python de QGIS, de manera que no s'indica la importació de mòduls i classes.
